Calibración de Nivel Freático – Ejemplo

Resumen ejecutivo: calibración NF usando piezómetros…
October 11, 2025

Este trabajo presenta un análisis conceptual de calibración del nivel freático usando piezómetros y una interpolación suave para construir una superficie NF coherente.

Contexto#

Se analizaron datos piezométricos históricos en un entorno minero.

Resultados#

  • Ajuste espacial consistente
  • Buen comportamiento en zonas con mayor densidad de datos

Conclusiones#

La metodología permite representar de forma coherente el comportamiento hidráulico.
